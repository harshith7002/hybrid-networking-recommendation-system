# Enigma26 Submission – CodeFest'26

## Folder Structure
- Notebooks/ : All training + modeling notebooks
- Models/    : Saved trained models and vectorizers
- Output/    : Final submission file

## Approach Summary
1. Notebook 01: Data understanding + cleaning
2. Notebook 02: TF-IDF vectorization for text columns
3. Notebook 03: CatBoost regression model trained on engineered features
4. Notebook 03B: SVD matrix factorization baseline for interaction scoring
5. Notebook 04: Final ensemble (CatBoost + SVD) and submission generation

## Final Output
The final predictions are saved in Output.
